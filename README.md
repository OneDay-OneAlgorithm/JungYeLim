## 🍺 Rule
- 하루에 한 문제 이상 풀기
- 매주 풀 문제 체크리스트✅ Issue 작성
- PR에 후기 남기기

## 🍺 Convention
### 파일명
-  `문제번호.확장자` 
-  ex) 1128.cpp

### Commit
  - 깃모지 사용
  - gitmoji + [문제번호] 문제 이름
  - ex) 📝 [1128] 너와 나의 알고리즘

### Issue
- `[주차] 기간`
- ex) [1회차] 230915 - 230917

### PR
- branch명 : `<주차>/<날짜>`
  - ex) 1주차/230915
- PR 제목 : `[주차] 날짜`
  - ex) [1주차] 230915
- 라벨 붙이기
